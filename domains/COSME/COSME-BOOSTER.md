## R&D 마일스톤 (7-verb · [PLAYBOOK.md](PLAYBOOK.md) 참조)

- [ ] spec: 시판 뷰티 디바이스 TOP-N (LG프라엘·메디큐브) + LED 파장·효능·분류 매핑
- [ ] spec: **분류 확정** — 화장품 vs 의료기기 (효능 표방 강도가 결정)
- [ ] structure: LED 파장 (적색 630nm·근적외 850nm·청색 415nm) + 세포 활성·항균 메커니즘
- [ ] design: 디바이스 설계 (파장 × 강도 × 시간 × 안전) + 부스터 화장품 동시 처방
- [ ] analyze: pilot head-to-head — 파장 검증 + 세포 활성 + 임상
- [ ] synthesize: GMP 시제품 제작 (분류별 GMP 기준)
- [ ] verify: 파장 정확도 ±5nm, 세포 활성 +20% vs 시판, 안전성 통과
- [ ] handoff: 분류별 신고/인증 + 양산 + 가정용 시술기 채널 + 구독 모델

## 차별 가설

> LED 파장 ±5nm 정확도 + 세포 활성 +20% + 부스터 화장품 동시 → **가정용 시술기 차별** · 구독·앱 연동

## 핵심 측정 지표

- LED 파장: spectrometer (±5nm)
- LED 강도: irradiance (mW/cm²)
- 세포 활성: in vitro fibroblast 증식
- 임상 (의료기기 시): 12주 효능 + IRB 임상
