> 캠페인 한 페이지 요약 — 누가 와도 5분 안에 흐름 파악 가능.

## 1. 7요소 도입

| 요소 | 내용 |
|---|---|
| 아이콘 | 🦋 |
| 이름 | `RTSC` (Room-Temperature Superconductor) |
| 별칭 | "상온초전도체 사냥" |

## 2. 큰 그림 (한 장 그림)

```
🎯 목표: 상온(293K=20°C) × 일반압(1기압)에서 초전도

   현실: 기존 모든 초전도체는 둘 중 하나 (혹은 둘 다) 못 만족
   ┌─────────────────────────────────────────────────────┐
   │   ❄️ 영하 (낮은 온도) 필요    ───→ 일상 못 씀       │
   │   🗜️ 초고압(150-200GPa)        ───→ 다이아몬드 압착기│
   │                                       바깥에선 못 씀  │
   └─────────────────────────────────────────────────────┘
   둘 다 풀려면 = 5축짜리 5D 코너 동시 만족
```

## 3. 5축짜리 5D 지도 (캠페인 핵심 도구)

```
   초전도가 일어나려면 다섯 가지 조건이 동시에 잘 맞아야:

   ┌──────────────────────────────────────────────────┐
   │ ① m-sign (탈출 여부)   m>0 = 우물 탈출           │
   │ ② ω_log (포논 진동수)  높을수록 Tc ↑             │
   │ ③ μ* (전자 반발)       낮을수록 좋음             │
   │ ④ 동적 안정성          격자가 안 무너져야        │
   │ ⑤ 압력 P              1기압이어야 일상 사용     │
   └──────────────────────────────────────────────────┘

   알려진 물질들은 *각자 1~2축만 빠짐*:

       CaH₆  (215K 측정) → ①②③④ ✓  ⑤만 ✗ (150GPa 필요)
       labeh8 (117K 측정) → ①③④⑤(저압) ✓  ②만 ✗ (ω_log 낮음)
       H₃S   (203K 측정) → ②③④ ✓     ①⑤ ✗

   → 빠진 단 1축을 닫으면 = RTSC!
```

## 4. 우리 전략 (4중 절단)

```
   캠페인이 동시에 시도하는 4가지 메커니즘 (모두 한 분자에서):

   🪤 ① M-Be-H precompressor
        가벼운 금속(Be)이 H를 *미리 압축* → 외부 압력 작아도 OK

   🪞 ② ZPE quantum cap
        H가 워낙 가벼워 *양자떨림*만으로 격자 안정

   ♾️ ③ SSCHA 고정점
        양자떨림 고려한 phonon 자체일관 풀이

   🔪 ④ chemical surgery
        soft-mode 자리에 정확히 cation 박아 안정화
```

```
   └─────────────────────────────────────────────────────────┘
   
   주력 2개:
   - CaH₆ 감압 게이트  → "그냥 압력 빼면 살아남나?"
   - sc2be2h6 vc-relax → "가벼운 cation 쓰면 ω_log↑되나?"
```

## 6. 깊은 통찰 (Cycle 13 발견)

```
   🐍 Cooper-Kramers ouroboros
   
   초전도의 "페어링" = 시간역 짝 (k↑, −k↓)
                        ↑ 자기 시간역 = 닫힌 고리
   
   → 초전도 자체가 시간이 자기를 마주보는 우로보러스
   → RTSC = 상온에서도 이 고리가 안 끊어지는 분자
```
