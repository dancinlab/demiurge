## 1. 💆 COSME-SCALP — structure + design

### 1-1. structure (활성성분 매트릭스)

#### A. 6대 자동허가 조합 (의무)

| 조합 | 성분 | 농도 | 분자 메커니즘 (stdlib §2) | 기여 |
|---|---|---|---|---|
| ① | 덱스판테놀 | 0.2% | 판테놀 → 판토텐산 → CoA → 모낭 대사 활성 | 두피 보습 |
| ① | 살리실릭애씨드 | 0.25% | 각질용해 → 모낭 sebum plug 제거 | 두피 환경 |
| ① | 엘-멘톨 | 0.3% | TRPM8 활성 → 쿨링 + 혈류 | 두피 자극 |
| ② | 나이아신아마이드 | 0.3% | NF-κB 억제 → 항염 | 두피 진정 |
| ② | 덱스판테놀 | 0.5% | (동일) | (동일) |
| ② | 비오틴 | 0.06% | 모낭 케라틴 합성 보조 | 모발 |
| ② | 징크피리치온액(50%) | 2.0% | Cu²⁺ 과축적·Fe-S 손상 (non-azole) | 항진균 보조 |

#### B. 보조 차별 성분 (시너지 후보)

| 후보 | 기능 | 메커니즘 | 임상 근거 (Agent 2) |
|---|---|---|---|
| 카페인 | 5α-환원효소 보조 차단 + 혈류 | 5αR 경쟁적 억제 + adenosine 차단 | 24주 모근 +5-10% (사례) |
| 펩타이드 (Procapil, Capixyl 등) | 모유두세포 활성 | DPC Wnt/β-catenin 회복 | 24주 모근 +5-15% |
| 식물성 엑소좀 (인삼·자스민) | DPC 증식 보조 | 세포 활성 신호전달 (인체세포배양액 가이드 준수) | in vitro DPC +30% |
| 아데노신 (기능성 인증 성분) | 모낭 혈류 + 모유두 활성 | A2 수용체 → cAMP 증가 | 기능성 화장품 인증 사례 |
| 자생식물 추출물 (당귀·천궁) | 항염 + 혈류 (한방) | 다성분 시너지 (보고 한계) | 임상 사례 부족 (한계) |

### 1-2. design (처방 설계 — 3 pilot 후보)

#### Pilot A: 자동허가 BASIC (1순위 진입)
- **조합**: ② 정량 (나이아신아마이드 0.3% + 덱스판테놀 0.5% + 비오틴 0.06% + 징크피리치온 2%) + pH 5.5 약산성 + 폴리쿼터늄-10
- **전달체계**: rinse-off 일반 (나노에멀젼 X — 비용↓)
- **클레임**: "탈모 증상 완화에 도움" (자료 면제)
- **진입**: 1-3개월·수백만원
- **차별**: 6대 정량 + 약산성 + 무실리콘

#### Pilot B: 자동허가 PLUS (1.5순위)
- **조합**: ② 정량 + **카페인 0.5% + 펩타이드 (Procapil) 3%**
- **전달체계**: 80nm 나노에멀젼 (활성성분 모낭 침투)
- **클레임**: "탈모 증상 완화에 도움" + 24주 임상 자체 데이터로 차별
- **진입**: 6-8개월·5,000-9,000만원 (24w TrichoScan)
- **차별 목표**: 24주 모근 +10%p (시판 6대 단독 대비)

#### Pilot C: 신규 효능 차별 (2순위 강화)
- **조합**: ② 정량 + 카페인 0.5% + 펩타이드 3% + **식물성 엑소좀 10⁹/mL**
- **전달체계**: 80nm 나노에멀젼 + 두피 마사지 도구 동봉
- **클레임**: "탈모 증상 완화에 도움" + 두피 마이크로바이옴 회복 (자체 임상)
- **진입**: 6-8개월·7,000-12,000만원 (24w TrichoScan + 16S × 2)
- **차별 목표**: 24주 모근 +15% + 마이크로바이옴 Shannon +0.5

### 1-3. 권고 (포그라운드 순차)

```
즉시 진입 → Pilot A (자료 면제·1-3개월·OEM 직발주)
            ↓
24주 임상 보강 → Pilot B (한 발 더 차별)
            ↓
프리미엄 라인업 → Pilot C (마이크로바이옴 차별)
```

## 2. 👁 QD-LENS — structure + design

### 2-1. structure (활성성분 매트릭스)

#### A. 보존제 (살균력)

| 활성성분 | 농도 | 메커니즘 (stdlib §2) | 안구 자극 |
|---|---|---|---|
| 폴리쿼터늄-1 (PQ-1) | 0.001% | 박테리아 막 4차암모늄 + 렌즈 흡수↓ | 낮음 |
| PHMB (폴리헥사메틸렌비구아니드) | 0.0001% | polycationic biguanide 막 결합·파괴 | 중간 (렌즈 축적) |
| 과산화수소 (H₂O₂) | 3% (별도 중화) | 산화 stress | 보존제 free (최저) |
| 클로르헥시딘 (CHX) | 0.005% | bis-biguanide | 높음 (사용 X) |

#### B. 세척 계면활성제

| 후보 | 기능 |
|---|---|
| 폴리옥사머 (Poloxamer 407) | 단백질 분해 보조 + 점도 |
| Tween 80 | 비이온성 세척 |
| 옥토페닐 폴리에톡시에탄올 (OPP) | 단백질 분해 |

#### C. 보습·완충

| 후보 | 기능 |
|---|---|
| 히알루론산 0.05% | 안구 점막 친화·보습 |
| 히드라네이트 | 보습 |
| 붕산·붕사 완충제 | pH 7.0±0.5 (안구 호환) |

#### D. 컬러렌즈 색소 보호 (화이트스페이스)

| 후보 | 기능 |
|---|---|
| EDTA 0.05% | 금속 이온 킬레이션 (TiO2 안정) |
| 시트르산 완충 | 산화 방지 |
| 어유 유래 토코페롤 | 항산화 (색소 산화 방지) |

### 2-2. design (처방 설계 — pilot 후보)

#### Pilot A: 다목적 표준 (1순위)
- **보존제**: PHMB 0.0001%
- **세척**: Poloxamer 407 1% + Tween 80 0.1%
- **보습**: 히알루론산 0.05%
- **완충**: 붕산·붕사 (pH 7.2, 290 mOsm)
- **차별**: 시판 동등 + **각막 세포독성 IC50 2배↑**
- **진입**: 3-4개월·1,000-2,000만원

#### Pilot B: 컬러렌즈 전용 (화이트스페이스 ★)
- **A + EDTA 0.05% + 토코페롤 0.01% + 시트르산 완충**
- **클레임**: "컬러렌즈 전용 — 색소 보호" (한국 컬러렌즈 USD 3.4억 시장)
- **차별**: 사용 30회 후 색차 ΔE ≤2.0 + Pilot A 효능 유지
- **진입**: 3-4개월·1,000-2,000만원 + 색차 시험 추가 100-300만원

#### Pilot C: 민감군 (H₂O₂ 시스템)
- **3% H₂O₂ + 별도 중화 케이스 (백금 코팅 디스크)**
- **차별**: 보존제 free·민감군 1위 (에이오셉 대안)
- **진입**: 6-8개월·2,500-5,000만원 (중화 시스템 GMP)

### 2-3. 권고

```
즉시 진입 → Pilot B (컬러렌즈 전용) — 화이트스페이스 + 한국 컬러렌즈 락인
            ↓
양산 후 라인업 확장 → Pilot A (다목적 표준)
            ↓
프리미엄 라인 → Pilot C (민감군)
```

## 3. 🌸 QD-PAD / QD-ECOPAD — structure + design

### 3-1. structure (활성성분 매트릭스)

#### A. 흡수체 (SAP — Super Absorbent Polymer)

| 후보 | 흡수량 | 친환경 | 비고 |
|---|---|---|---|
| 폴리아크릴산 SAP (석유 유래) | 매우 높음 | ✗ | 시판 대다수 |
| 셀룰로오스 SAP (식물 유래) | 중간-높음 | ✓ | 친환경 라인 (해피문데이 등) |
| 키토산 SAP | 중간 | ✓ | 항균 보조 |
| 펄프 + SAP 혼합 | 가변 | △ | 일반형 |

#### B. 표면시트 (톱시트)

| 후보 | 자극 | 친환경 |
|---|---|---|
| 부직포 (PP) | 중간 | ✗ |
| 코튼 (일반) | 낮음 | △ |
| 유기농 코튼 (GOTS) | 매우 낮음 | ✓ |
| 셀룰로오스 부직포 | 낮음 | ✓ |

#### C. 항균·소취 보조

| 후보 | 기능 | 메커니즘 |
|---|---|---|
| 키토산 (자연 항균) | 광범위 항균 + 락토바실러스 보호 | 박테리아 막 결합 + 양친매성 |
| 은나노 (실버나노) | 광범위 항균 | Ag⁺ 방출 → 박테리아 막 손상 |
| 활성탄 (소취) | 흡착 | 분자 흡착 |
| 차나무 오일 (천연) | 항균·항진균 | 막 손상 |

#### D. 백시트 (생분해)

| 후보 | 생분해 | 비고 |
|---|---|---|
| PE (폴리에틸렌) | ✗ | 시판 대다수 |
| PLA (폴리락트산) | 180일 | 친환경 |
| 셀룰로오스 백시트 | 90-180일 | 친환경 |

### 3-2. design (처방 설계 — pilot 후보)

#### Pilot A: QD-PAD BASIC (1순위 일반형)
- **흡수체**: 펄프 + 폴리아크릴산 SAP
- **표면**: 코튼 부직포
- **항균**: 키토산 코팅
- **백시트**: PE
- **차별**: VOC 84종 ≤시판 1/2 + 키토산 항균
- **진입**: 2-3개월·1,500-3,000만원

#### Pilot B: QD-ECOPAD (1순위 친환경 ★)
- **흡수체**: 셀룰로오스 SAP + 펄프
- **표면**: 유기농 코튼 (GOTS)
- **항균**: 키토산
- **백시트**: PLA (생분해 180일)
- **차별**: VOC ≤일반 시판 1/3 + 생분해 180일 + 비건 인증
- **진입**: 2-3개월·1,500-3,000만원 + 인증 비용 500만원
- **타겟**: mid-price 9-11k 슬롯 (해피문데이 14k vs 일반 7k)

#### Pilot C: 초민감군 (피부 트러블)
- **흡수체**: 셀룰로오스 SAP
- **표면**: 의료 등급 부직포 + 시카 코팅
- **항균**: 키토산 + 차나무 오일
- **백시트**: PLA
- **차별**: 피부 트러블 환자용 (skin patch test 강화)
- **진입**: 3-4개월·2,500-4,000만원

### 3-3. 권고

```
즉시 진입 → Pilot B (ECOPAD) — 친환경 + mid-price 미충족 슬롯 + 인증 차별
            ↓
양산 후 라인업 확장 → Pilot A (BASIC) — 일반형 가성비
            ↓
프리미엄 라인 → Pilot C (초민감군)
```
